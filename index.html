    Welcome to SQLviz.

    SQLViz is an easy and open source way to turn SQL into graphical visualization of data within your database. SQLViz supports Postgres and Redshift. It allows for rapid analysis and dissemination of insights locked into your data sets.

    $ git checkout https://github.com/sqlviz/sqlviz.git
$ sqlviz/deploy_script/deploy_first_time.sh

    Charts

    Charts are the basic element of visualizations. Each chart is powered by 1 SQL statement.

    Dashboards

    Dashboards are combinations of multiple charts.

    Email Reports

    Dashboards can be emailed on a set schedule.
